# React Alternatives — Fluent React

這份簡報探討 React 生態圈中的替代方案（例如 Vue、Angular、Solid、Svelte、Qwik 等），介紹各框架的反應性、效能與開發者體驗上的差異。

[簡報連結](https://weilocus.github.io/fluent-react/react-alternatives/)

資料夾內容

- `slides.md`：簡報主要內容與投影片內容。
- 其他環境設定檔（`package.json`, `netlify.toml`, ...）用於本地開發與部署

本地開發

1. 安裝套件（可使用 pnpm 或 npm）：

```
pnpm install
# or
npm install
```

2. 啟動：

```
pnpm dev
# or
npm run dev
```

3. 開啟瀏覽：

http://localhost:3030

修改簡報，請編輯 `slides.md` 並重新整理瀏覽器。

閱讀建議

- 建議以深色模式（dark mode）閱讀；本簡報預設套用 `class: bg-neutral-800 text-white`，在淺色主題下對比可能不足。
