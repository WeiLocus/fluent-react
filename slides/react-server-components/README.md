# React Server Components — Fluent React

這份簡報介紹 React Server Components（RSC）的伺服器組件、用戶端組件概念、序列化的重要性、簡介伺服器操作，讓我們更了解 RSC 如何做到 server rendering。

[簡報連結](https://weilocus.github.io/fluent-react/react-server-components/)

資料夾內容

- `slides.md`：簡報內容與投影片。
- 其他環境設定檔（`package.json`, `netlify.toml`, ...）用於本地開發與部署

本地開發

1. 安裝套件（可使用 pnpm 或 npm）：

```
pnpm install
# or
npm install
```

2. 啟動：

```
pnpm dev
# or
npm run dev
```

3. 開啟瀏覽：

http://localhost:3030

修改簡報，請編輯 `slides.md` 並重新整理瀏覽器。

閱讀建議

- 建議以深色模式（dark mode）閱讀；本簡報預設套用 `class: bg-neutral-800 text-white`，在淺色主題下對比可能不足。
