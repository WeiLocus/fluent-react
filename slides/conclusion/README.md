# Conclusion — Fluent React

這份簡報回顧 Fluent React 書中重要的觀點與結論，包含對於設計取捨、效能考量與未來方向的建議。

資料夾內容

- `slides.md`：主要內容在此編輯
- 其他環境設定檔（`package.json`, `netlify.toml`, ...）用於本地開發與部署

本地開發

1. 安裝套件（可使用 pnpm 或 npm）：

```
pnpm install
# or
npm install
```

2. 啟動：

```
pnpm dev
# or
npm run dev
```

3. 開啟瀏覽：

http://localhost:3030

修改簡報，請編輯 `slides.md` 並重新整理瀏覽器。

閱讀建議

- 建議以深色模式（dark mode）閱讀；本簡報預設套用 `class: bg-neutral-800 text-white`，在淺色主題下對比可能不足。
